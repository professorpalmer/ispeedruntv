About iSpeedrun.TV :

Update April 8th, 2023 : Account creation is now up and running. Discord chat integration is now accessible!

iSpeedrun.TV is a project created by myself, Cary Palmer, utilizing ChatGPT to accelerate the process of what I could do alone. Content is all based on my own sourced ideas, prompts, etc.

I created my own site where I can share what I want, when I want, and see what I can teach myself in the process.

This site is a work in progress and will be for a while. Expect to see it go up, down, and change a lot. If you're interested in contributing to the development of the site, feel free to reach out!
